## Tech Stack

| Layer | Technology |
|---|---|
| Markup | HTML5 |
| Styling | Tailwind CSS (CDN) + custom CSS (`assets/css/style.css`) |
| Typography | Inter (Google Fonts) |
| Scripting | Vanilla JS (`assets/js/scripts.js`) |
| Theme | CSS custom properties, dark/light mode toggle |
| Hosting | GitHub Pages (custom domain via CNAME) |
